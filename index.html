My Skills

  • HTML5

    É uma linguagem de marcação de hipertexto para apresentar e estruturar o conteúdo na web.

  • CSS

    De forma prática, o CSS funciona como uma camada de personalização ao conteúdo visível.

  • Javascript

    É uma linguagem com tipagem fraca, ou seja, não permite que o programador defina o tipo de variável.

  • React

    É uma biblioteca modular, o que significa que os componentes podem ser facilmente reutilizados e compartilhados entre diferentes partes da aplicação.

  • Typescript

    É uma linguagem de código aberto desenvolvida pela Microsoft que foi construída em cima do Javascript.

  • React-Router-Dom

    É uma biblioteca para o React criada com o objetivo de resolver problemas de rotas para as páginas de um site.

  • Styled-Components

    É uma biblioteca do React e do React Native que nos permite criar componentes de estilo ao escrever códigos CSS em um arquivo JavaScript.

  • Axios

    É um cliente HTTP baseado em Promises para fazer requisições. Pode ser utilizado tanto no navegador quanto no Node. js ou qualquer serviço de API.

  • Redux

    É uma biblioteca feita para gerenciar estados em uma aplicação, onde um componente terá um estado que poderá ser usado por vários outros componentes.

  • Context Api

    É uma opção simples para compartilhar dados entre componentes que não estão diretamente na árvore de componentes.

  • Git

    Git é o sistema de controle de versão usado para controlar o histórico de alterações de arquivos.

  • Github

    É uma plataforma para hospedagem e controle de versão de código utilizando o sistema Git como base.
